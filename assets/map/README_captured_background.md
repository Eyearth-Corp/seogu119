# 캡처 배경 이미지 사용 방법

## 파일 추가 방법

1. 캡처한 배경 이미지를 준비합니다.
2. 이미지를 `captured_background.webp` 파일명으로 저장합니다.
3. 이 폴더 (`assets/map/`)에 파일을 추가합니다.

## 이미지 요구사항

- **파일명**: `captured_background.webp`
- **형식**: WebP (권장) 또는 PNG/JPG도 가능
- **크기**: 2403 x 2900 픽셀 (지도와 동일한 크기)
- **최적화**: 파일 크기를 줄이기 위해 WebP 형식 사용 권장

## 사용 방법

1. 이미지 파일을 추가한 후 Flutter 앱을 재시작합니다.
2. 지도 하단의 레전드에서 "캡처배경" 버튼을 클릭합니다.
3. 버튼이 활성화되면 캡처한 배경이 70% 투명도로 표시됩니다.

## 투명도 조절

투명도를 변경하려면 `map_widget.dart` 파일의 292번째 줄에서 `opacity` 값을 수정하세요:
```dart
opacity: _showCapturedBackground ? 0.7 : 0.0,  // 0.7 = 70% 불투명도
```

## 참고사항

- 파일이 없어도 앱은 정상 작동합니다 (errorBuilder 처리됨)
- 캡처 배경은 기본 지도 위, 지형 레이어 아래에 표시됩니다.
